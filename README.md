# MagMoment AR Engine v1

This is a web-based AR engine built using MindAR for MagMoment products.

## How it works
1. User scans QR code
2. Camera opens in browser
3. Printed photo/magnet is detected
4. Video memory plays inside image

## Requirements
- HTTPS hosting (GitHub Pages / Netlify / Cloudflare)
- Chrome / Safari mobile browser
- High-contrast image targets

## Folder notes
- assets/targets.mind → compiled image target
- assets/video.mp4 → customer memory video
- assets/logo.png → brand watermark
- assets/chime.mp3 → optional sound

## Recommended video specs
- Ratio: 3:4
- Length: 8–12 seconds
- Format: MP4 (H.264)

## Deployment
Upload entire folder to any static hosting.
